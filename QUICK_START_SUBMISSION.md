## Part 2: GitHub Marketplace (10 mins)
1. Go to: https://github.com/settings/apps
2. Click "New GitHub App"
3. Copy/paste from MARKETPLACE_SUBMISSION_GITHUB.md
4. Go to: https://github.com/marketplace/new
5. Upload logo + screenshots
6. Set pricing (4 tiers)
7. Click Submit
8. ✅ Done! Wait 2-5 days for approval
