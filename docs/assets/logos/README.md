# DSG Logo Assets

Official logo files for DSG (Deterministic Execution & Governance).

## Logo Variations

### Primary Logo
- **File:** `dsg-primary-logo.png` (291 KB)
- **Usage:** Marketing materials, website headers, presentations, landing pages
- **Minimum Size:** 120px width
- **Clear Space:** 1/4 of logo width on all sides
- **Backgrounds:** White, light gray, or navy
- **Contains:** "DSG" text with hexagonal badge (Navy, Gold, Red accent)

### App Icon
- **File:** `dsg-app-icon.png` (110 KB)
- **Usage:** App store listings, app headers, favicon, browser tabs
- **Dimensions:** Square format (recommended 1024x1024px)
- **Minimum Size:** 32px (favicon), 180px (app store)
- **Colors:** Navy background with white hexagon, gold accent, red accent

### Monogram Symbol
- **File:** `dsg-monogram-symbol.png` (120 KB)
- **Usage:** Favicon, small icons, app badges, social media, avatar
- **Minimum Size:** 32px
- **Clear Space:** Equal to symbol width
- **Contains:** Hexagonal badge with circuit board elements (white, gold, red on navy)
- **Variations:** 
  - Color (Primary) - Navy background with accents
  - Monochrome (Navy #001F3F) - Navy only, no background
  - Inverted (White on navy) - For dark backgrounds

### Monochrome Variation
- **File:** `dsg-monochrome-variation.png` (80 KB)
- **Usage:** Print, PDFs, restricted color environments, single-color applications
- **Color:** Navy (#001F3F) only
- **Background:** White or light gray
- **Minimum Size:** 120px width
- **Contains:** Monogram symbol + "DSG" text in navy

## Export Formats

All logos should be exported in:
- **SVG** — Vector format for web (scalable, smallest file size)
- **PNG** — 300dpi for print, 72dpi for web
- **PDF** — For print materials (press release, documents)
