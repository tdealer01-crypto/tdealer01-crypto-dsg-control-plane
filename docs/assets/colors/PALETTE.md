## Color Accessibility

- **Contrast Ratios:** All text meets WCAG AA standard (4.5:1 for normal, 3:1 for large)
- **Colorblind Safe:** Palette tested with protanopia and deuteranopia
- **Dark Mode:** All colors have corresponding dark-mode variants

## Usage Guidelines

- **Web:** Use hex codes (#001F3F)
- **Mobile:** Use RGB values (0, 31, 63)
- **Print:** Use CMYK values (100, 92, 78, 75)
- **Design Systems:** Reference via CSS variables: `--dsg-navy`, `--dsg-red`, etc.
