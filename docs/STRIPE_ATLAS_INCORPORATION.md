## ✅ Atlas Special Features

### **1. 83(b) Election** (Automatic)
```
What: IRS form filed automatically
Why: Starts your vesting clock
Effect: Prevents future tax complications
Timeline: Filed within 30 days of share issuance

✅ Atlas handles automatically
✅ Recommended by all startup lawyers
✅ Expected by investors
```

### **2. IP Assignment Agreement**
```
What: You assign IP → Company owns IP
Why: Important for investors/credibility
How: Payment is via stock (6 million shares)
Fair Market Value: Standard $100 USD

⚠️ Check if you have valuable IP:
   - Patents?
   - Significant prior inventions?
   - Tech developed pre-incorporation?

If YES with >$100 value:
→ Consider consulting lawyer
→ May need IP assignment clause
```

### **3. Registered Agent Service**
```
What: Receives official Delaware notices
Cost: $500 (included in Atlas fee) for year 1
      $100/year after
Who: Stripe Atlas provides agent

Automatic renewal:
✅ Each year until you cancel
✅ Need to maintain for Delaware compliance
```

### **4. EIN (Employer Identification Number)**
```
Timeline: 1-2 business days (with SSN + address + phone)
How: Atlas files Form SS-4 + Form 8821
Delivery: Digital copy via Atlas + Physical copy to address
Use: Stripe Payments, bank account, taxes, employees
```
