## 3. DSG ONE คืออะไร (5 นาที)

> "DSG ONE คือ AI Control Plane — ศูนย์กลางควบคุม AI"

ไม่ใช่แค่ "ChatGPT สำหรับองค์กร"

เป็นระบบที่:
1. **เชื่อมทุกช่องทาง** → Salesforce / Zendesk / Slack / LINE / Email เข้าไว้ด้วยกัน
2. **ควบคุมทุก AI Action** → Policy Gate ตรวจสอบทุกการกระทำ
3. **บันทึกทุกอย่างอัตโนมัติ** → Audit Trail ครบถ้วน
4. **ทำงาน 24/7** → AI Agent ทำงานแทนคนตลอดเวลา

**Slide: Architecture ภาพรวม**

## 6. Use Case ที่ลูกค้าสนใจ (5 นาที)

### Use Case 1: Support Automation
- ลูกค้าส่งคำถามมาทาง LINE/Email
- AI Agent ตอบกลับอัตโนมัติ 80%
- ที่เหลือส่งต่อทีมงานพร้อมข้อมูลครบถ้วน
- **ผลลัพธ์: ลดเวลาตอบจาก 1 วัน → 15 นาที**

### Use Case 2: Sales Enablement
- AI Agent ดึงข้อมูลจาก Salesforce + Zendesk
- สรุปประวัติลูกค้าทั้งหมดใน 1 วินาที
- ทีมขายไม่ต้องค้นหาข้อมูลเอง
- **ผลลัพธ์: ทีมขายปิด deal ได้เร็วขึ้น 3 เท่า**

### Use Case 3: Compliance Automation
- ทุก AI Action ถูกบันทึกอัตโนมัติ
- Audit Trail ครบถ้วน ไม่ต้องทำเอง
- EU AI Act / ISO 42001 ready
- **ผลลัพธ์: ประหยัดเวลา compliance audit 90%**
