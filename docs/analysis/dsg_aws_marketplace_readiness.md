## 6. Pricing Model for Marketplace

### Option A: Listing as SaaS (Recommended)
- **How it works:** DSG charges via Stripe; AWS Marketplace seats metering
- **Pricing:** ฿490/month + $0.10 per governed execution
- **Revenue split:** 70% DSG, 30% AWS (standard)
- **Advantage:** Flexible billing, AWS handles payment
- **Disadvantage:** AWS takes 30% cut

### Option B: Professional Services
- **How it works:** AWS Marketplace lists as consulting engagement
- **Pricing:** $5K-$25K per deployment
- **Revenue split:** 70% DSG, 30% AWS
- **Advantage:** Higher margins per deal
- **Disadvantage:** Slower sales cycle

### Option C: Bring-Your-Own-License (BYOL)
- **How it works:** Customer pays DSG directly, AWS just lists
- **Pricing:** Direct Stripe billing (no AWS cut)
- **Revenue split:** 100% DSG
- **Advantage:** No AWS margin loss
- **Disadvantage:** Lower visibility on marketplace

**Recommendation:** Option A (SaaS) — fastest to market, AWS handles billing
