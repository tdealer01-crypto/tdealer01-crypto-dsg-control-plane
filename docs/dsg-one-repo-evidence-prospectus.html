README.md

README ระบุชัดว่า DSG ONE เป็น enterprise runtime control plane for AI operations ที่ทำให้ AI execution ไม่เป็น black box และเพิ่มความสามารถด้าน authenticated operator access, organization-scoped workspaces, agent creation, execution visibility, audit-oriented runtime evidence, mission/readiness surfaces, usage/quota/billing awareness และ proof surfaces แบบ public กับ verified.

ธีสหลักของรีโปคือ ถ้า AI จะทำงานใน production ได้ ระบบต้อง controllable, inspectable, และ reviewable แบบ real time.

The README describes DSG ONE as an enterprise runtime control plane for AI operations that prevents execution from being treated as a black box. It lists authenticated operator access, organization-scoped workspaces, agent creation, execution visibility, audit-oriented runtime evidence, mission/readiness surfaces, usage/quota/billing awareness, and separate public versus verified proof surfaces.

The repo thesis is simple: if AI is used in production, it must be controllable, inspectable, and reviewable in real time.

Execute Route Facts

Spine Execute

  • Route: /api/spine/execute and /api/execute re-exports into it
  • Rate limit: 60 requests per 60 seconds
  • Authorization: requires Bearer token
  • Payload contract: agent_id is required
  • Access control: enforced by org role checks before execution
  • Response behavior: returns rate-limit headers with result
