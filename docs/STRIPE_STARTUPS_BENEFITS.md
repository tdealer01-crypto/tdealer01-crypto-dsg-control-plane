## 🎯 สิ่งที่สามารถใช้เครดิตได้

เครดิต Stripe Startups สามารถหักลดค่าธรรมเนียม Stripe **ทั้งหมด** ได้:

### 1. **Payments** (ใช้ได้ - ระบบปัจจุบัน)
- Stripe Checkout (การจ่ายเงินแบบครั้งเดียว)
- Payment Links
- Stripe.js / Elements
- **Discount:** ~70% off standard transaction fees
- **ระบบ DSG ใช้:** ✅ `/api/billing/checkout`

### 2. **Billing** (ใช้ได้ - ระบบปัจจุบัน)
- Subscriptions & recurring billing
- Metered billing (usage-based)
- Trial periods
- Invoicing
- **Discount:** Applied to subscription fees and overage charges
- **ระบบ DSG ใช้:** ✅ Plans (pro, business, enterprise) + Skills bundles

### 3. **Connect** (ใช้ได้ - ระบบปัจจุบัน)
- Connected accounts for marketplaces
- Payouts to sellers
- Onboarding
- Reporting
- **Discount:** Applied to Connect fees
- **ระบบ DSG ใช้:** ✅ OAuth integration + Stripe App

### 4. **Radar** (ปรับปรุง)
- Advanced fraud detection
- Machine learning models
- **Cost:** Normally $0.01-0.05 per transaction
- **ด้วยเครดิต:** Fully covered

### 5. **Terminal** (ไม่ใช้ปัจจุบัน)
- In-person payments with Stripe Terminal readers
- **Discount:** Applicable to transaction fees

### 6. **Tax** (ไม่ใช้ปัจจุบัน)
- Automated tax calculation
- Tax reporting
- **Cost:** $0.01 per calculation
- **ด้วยเครดิต:** Fully covered

### 7. **Treasury** (ไม่ใช้ปัจจุบัน)
- Financial accounts
- Transfers
- **Cost:** Varies by feature
- **ด้วยเครดิต:** Applicable

### 8. **Issuing** (ไม่ใช้ปัจจุบัน)
- Virtual and physical cards
- **Cost:** Per-card and transaction fees
- **ด้วยเครดิต:** Applicable

### 9. **Identity** (ไม่ใช้ปัจจุบัน)
- KYC/AML verification
- Document verification
- **Cost:** $0.25-2.00 per verification
- **ด้วยเครดิต:** Fully covered
