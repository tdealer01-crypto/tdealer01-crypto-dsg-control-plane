### Description

DSG Agent Governance lets you control AI agents before they modify your codebase.

**The problem:** Claude Code, OpenHands, Aider, and other coding agents can propose changes to your files. You want to review and approve before execution. There's no built-in gate.

**Our solution:** DSG installs as a GitHub App. Every time an AI agent proposes changes:
1. We intercept the proposal
2. Show you exactly what will happen
3. You approve or reject in 1 click
4. Every decision is logged with cryptographic proof

Perfect for developers using Claude Code, teams needing compliance, and enterprises managing AI agents at scale.

**What's included:**
- 🎯 GitHub App for frictionless setup
- ✅ Approval workflow with 24h auto-reject
- 🔍 Audit trail with SHA-256 proofs
- 📦 npm SDK for custom integrations
- 🚀 Freemium: 10 gate decisions/month

**Try free:** Install the app. Approved in 30 seconds.
