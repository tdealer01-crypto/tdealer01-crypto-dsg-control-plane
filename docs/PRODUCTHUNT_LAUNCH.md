## Description (2-3 sentences)
DSG ONE is a control plane that gates AI agents before execution. See every code change your AI assistant proposes. Approve or reject in seconds. Every decision logged with cryptographic proof.

Perfect for:
- Teams using Claude Code, OpenHands, Aider
- Enterprise governance + compliance
- Finance, healthcare, regulated industries

---

## Key Features (Bullet List for PH)
- 🎯 **Gate AI Agents** — Control Claude Code, OpenHands, Aider before they modify code
- ✅ **Approval Workflows** — Approve/reject changes in seconds. Automatic 24h rejection.
- 🔍 **Audit Trail** — Every decision logged with SHA-256 proof. Exportable JSON.
- 📋 **Compliance Ready** — EU AI Act, ISO 42001, NIST RMF evidence export
- 🚀 **Works Everywhere** — GitHub App, npm SDK, Next.js middleware, Vercel Edge
- 🔐 **Zero External Solver** — Pure TypeScript. Fast. Deterministic.
