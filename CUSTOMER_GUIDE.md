## ❓ FAQs

### Q: Is my data secure?
**A:** Yes! 
- JWT Bearer token authentication
- Encrypted credentials
- Row-level security (org-scoped data only)
- No data shared between organizations

### Q: What's the latency?
**A:** < 1 second for most operations
- Quick actions: typically <200ms
- Chat: 500ms delay + execution time
- Execution: ~1-10ms

### Q: Can I use this on mobile?
**A:** Yes! Full responsive design
- All buttons clickable
- Chat works great on phone
- Results display properly

### Q: What if I see an error?
**A:** Try these steps:
1. Refresh the page (Cmd+Shift+R)
2. Clear browser cache
3. Try logging out and back in
4. Contact support if issue persists

### Q: How do I logout?
**A:** Click the [Logout] button in top right

### Q: Can multiple users use this?
**A:** Yes! Each user logs in separately
- Each sees their own org data
- All sessions are independent
- No shared data between users
