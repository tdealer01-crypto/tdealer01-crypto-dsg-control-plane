# Marketplace Assets

Logos and screenshots for GitHub and Stripe App marketplace listings.

## Files Required

### Logo
- **File:** `logo.png`
- **Size:** 100x100 pixels
- **Format:** PNG, square aspect ratio
- **Usage:** GitHub Marketplace, Stripe App, Claude Code marketplace

### Screenshots
- **Format:** PNG, 1280x720 pixels (16:9 aspect ratio)
- **Count:** 5 screenshots recommended
- **Files:**
  - `screenshot-1-policy-gate.png` — Gate evaluation interface
  - `screenshot-2-replay-evidence.png` — Replay governance and proof
  - `screenshot-3-compliance-evidence.png` — Compliance evidence pack
  - `screenshot-4-runtime-governance.png` — Runtime lifecycle management
  - `screenshot-5-hermes-executor.png` — Hermes controlled executor

## Generation

Logo and screenshots are currently placeholders. For production launch:

1. Use design tools (Figma, Canva, etc.) to create professional marketing assets
2. Include DSG branding and key feature callouts
3. Use consistent color scheme: DSG blue (#0066CC) + white + dark gray
4. Save in this directory with exact filenames above
5. Verify dimensions before uploading to marketplaces

## Marketplace Upload URLs

- **GitHub Marketplace:** https://github.com/marketplace/new
- **Stripe App Marketplace:** https://dashboard.stripe.com/apps
- **Claude Code Plugin Marketplace:** https://claude.ai/plugins

## Asset Guidelines

### GitHub Marketplace
- Logo must be square (100x100px)
- Use clear, recognizable design
- Screenshots should show actual UI, not mockups
- Include descriptive captions

### Stripe App Marketplace
- Similar requirements to GitHub
- Focus on integration points with Stripe
- Show payment/billing-specific workflows

### Claude Code
- Logo should work at various sizes (16px to 256px)
- Screenshots should show Claude Code interaction
- Highlight governance/compliance benefits
